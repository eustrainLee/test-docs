# 安装说明

## 系统要求

- Node.js 16+
- npm 7+

## 安装步骤

1. 克隆仓库：
   ```bash
   git clone https://github.com/your-org/talisman.git
   ```

2. 安装依赖：
   ```bash
   cd talisman
   npm install
   ```

3. 配置环境：
   ```bash
   cp .env.example .env
   ```
